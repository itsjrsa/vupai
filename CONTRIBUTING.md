# Contributing

Thanks for your interest in vupai.

This project is open source (MIT) but is **not currently accepting external
contributions**. Pull requests opened against this repository will be closed
without review. This policy may change in the future.

## Bug reports

Bug reports are welcome via [GitHub Issues](https://github.com/itsjrsa/vupai/issues).
Please include your macOS version, how you installed vupai, and the exact steps
to reproduce.

## Security

Do not report security issues in public issues. See [SECURITY.md](SECURITY.md).
